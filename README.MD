# This project describes a Flux bug with custom
## Project overview
1. File [Main](./src/main/java/org/example/Main.java) contains the method that produces Flux with custom publisher
2. File [CustomRunnable.java](./src/main/java/org/example/CustomRunnable.java) represents as some generator of the data to a `Subscriber`.
3. File [AdapterProcessor](./src/main/java/org/example/AdapterProcessor.java) represents as Processor that consumes data and publish that to other publisher
4. File [MainTest.java](./src/test/java/org/example/MainTest.java) contains tests related to bugs

```shell
./gradlew clean build
```
